# Personal Portfolio Website 🌐

A clean, responsive personal portfolio website built using **HTML**, **CSS**, and **CSS Media Queries**.  
This project showcases my skills, projects, and contact information in a simple and user-friendly layout that works across desktop, tablet, and mobile devices.

---

## 🚀 Features

- Fully responsive design (mobile-first)
- Clean and modern UI
- Smooth navigation
- Cross-browser compatible
- Lightweight and fast loading
- Easy to customize and extend

---

## 🛠️ Built With

- **HTML5** – Structure and content  
- **CSS3** – Styling and layout  
- **Media Queries** – Mobile responsiveness  
- **Flexbox / Grid** – Layout management  

---

## 📱 Responsive Design

The website is optimized for multiple screen sizes using CSS media queries:

- Desktop 💻
- Tablet 📱
- Mobile 📲

---


✏️ Customization

Update content in index.html

Modify styles in style.css

Adjust breakpoints in media queries for different screen sizes

Replace images and in the folder
